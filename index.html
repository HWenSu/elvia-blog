Hexo 架站筆記

前言

紀錄用Hexo架設第一個靜態部落格網站

Why Hexo?

Hexo 具有下列幾項特點：

  • 基於 Node.js：Hexo 是一個使用 Node.js 開發的網誌框架。
  • 支持 Markdown：使用 Markdown 語法撰寫文章，方便且高效。
  • 快速生成：具有高效的生成速度，可以迅速將撰寫的內容轉換成靜態網頁。
  • 豐富的主題和插件：擁有豐富的主題庫和插件，可自定義博客的外觀和功能。
  • 一鍵部署：支持一鍵部署到多個平台，如 GitHub Pages、Heroku 等，方便快捷。

Get Started

  • 確認已安裝好 Git / node.js / npm

    → Hexo 官方

    → 在終端機上輸入指令：
    $ npm install -g hexo-cli

建立環境

  1. 前往指令：cd
  2. 建立 Hexo 部落格：hexo init 你的網站名稱

  1. 開啟 Hexo 伺服器：hexo server
  2. 新增文章: hexo new 文章名稱

全站配置

可以在 _config.yml 或 替代配置檔 中修改網站配置。

更改的資料沒有顯示在畫面中

hexo clean 清空暫存資料

hexo server 重新啟動

部屬到網頁需要的資料夾

hexo generate 產生 public 資料夾

hexo deploy

客製化頁面

hexo new page about

source 的資料夾裡面會多一個 about 的頁面

全域網站加入客製選單

執行以下:

git clone --depth 1 [https://github.com/hexojs/hexo-theme-landscape](https://github.com/hexojs/hexo-theme-landscape) themes/landscape

下載完會看到 theme 資料夾裡有 landscape資料夾，裡面有一個主題設定的 config.yml 檔
